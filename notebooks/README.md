# Notebooks

Esta pasta concentra os notebooks executáveis do projeto.

## Ordem Recomendada

| Semana | Notebook | Finalidade |
|--------|----------|------------|
| Semana 1 | `Semana_1_EDA_e_Preparacao.ipynb` | Análise exploratória, limpeza e preparação inicial |
