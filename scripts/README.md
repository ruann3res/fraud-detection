# Scripts

Pasta reservada para funções Python reutilizáveis.

No momento, a implementação principal está nos notebooks. Conforme o código se repetir, mover para scripts como:

- `preprocessing.py`: leitura, limpeza, seleção de atributos e escala;
- `clustering.py`: K-Means, DBSCAN e funções de comparação;
- `metrics.py`: métricas de clustering e classificação;
- `features.py`: geração de variáveis para Inteligência Computacional.

## Quando criar scripts

Criar scripts quando uma função for usada em mais de um notebook ou quando o relatório final precisar de pipeline reprodutível.
